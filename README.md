# Navalha Prime Barber Club — Demo Olegario Tech

Demo premium criada para apresentar a experiência que a Olegario Tech pode entregar para barbearias da região de Valinhos/Campinas.

A ideia central:

> O AppBarber/Trinks/WhatsApp cuida da agenda. A Olegario Tech cria a primeira impressão premium que faz o cliente querer agendar.

## Status

Estrutura inicial pronta em HTML, CSS e JavaScript puro, com GSAP e Lenis via CDN.

## O que já vem pronto

- Hero cinematográfico
- CTA de agendamento
- Simulador visual de agenda
- Serviços com curadoria
- Barbeiros fictícios
- Assinaturas
- Galeria
- Prova social
- Unidades fictícias em Valinhos/SP
- CTA comercial da Olegario Tech
- Estrutura pronta para Netlify

## Importante

Este é um projeto demonstrativo.

Nome, profissionais, avaliações, telefone e endereços são fictícios.  
Não usar nomes, logos, endereços ou dados reais de barbearias existentes sem autorização.

## Onde editar os dados

Edite:

```txt
src/data/content.js
```

Ali ficam links, serviços, barbeiros, planos, depoimentos e unidades.
